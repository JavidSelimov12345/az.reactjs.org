### JSX-də kommentləri necə yazmalıyam? {#how-can-i-write-comments-in-jsx}

```jsx
<div>
  {/* Komment bura yazılır */}
  Salam {name}!
</div>
```

```jsx
<div>
  {/* Çox sətrli kommentlər də
  əlavə etmək mümkündür. */}
  Salam {name}! 
</div>
```
