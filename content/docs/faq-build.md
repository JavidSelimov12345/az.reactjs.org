### JSX-də kommentləri necə yazmaq lazımdır? {#how-can-i-write-comments-in-jsx}

```jsx
<div>
  {/* Komment bura gedir */}
  Salam {name}!
</div>
```

```jsx
<div>
  {/* Çox sətrli kommentlər 
  ilə də işlətmək mümkündür. */}
  Salam {name}! 
</div>
```
